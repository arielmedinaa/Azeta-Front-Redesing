# Azeta Vite Project

Proyecto de Grupo Azeta migrado a Vite con estructura organizada.

## Estructura del Proyecto

```
src/
├── components/          # Componentes de React
│   ├── AzetaNosotros.jsx
│   ├── Navigation.jsx
│   ├── Hero.jsx
│   ├── About.jsx
│   ├── Stats.jsx
│   ├── Sectors.jsx
│   ├── Manifesto.jsx
│   ├── Footer.jsx
│   └── Cursor.jsx
├── data/               # Datos y constantes
│   └── constants.js
├── hooks/              # Hooks personalizados
│   └── useGSAP.js
├── styles/             # Estilos CSS
│   └── global.css
├── utils/              # Utilidades
│   └── SplitText.js
└── main.jsx           # Punto de entrada
```

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Características

- ✅ Migrado a Vite
- ✅ Estructura de componentes modular
- ✅ Estilos separados en CSS
- ✅ Datos centralizados
- ✅ Hooks personalizados
- ✅ Animaciones con GSAP
- ✅ Cursor personalizado
- ✅ Diseño responsivo
